# 어조 특성: 권위적 (Authoritative Tone)

## 개요

권위적 어조는 확정적 언어, 강한 주장, 전문가 포지셔닝을 통해 전문성과 확신을 전달한다. 필자를 해당 주제에 대한 지식 있는 권위자로 확립한다.

## 언어적 표지

### 확정적 언어

강한 진술:
- "~이다" (~인 것 같다 대신)
- "~해야 한다" (~하는 게 좋다 대신)
- "~이 정답이다" (~이 답일 수 있다 대신)

예시:
"이벤트 소싱은 분산 시스템에서 데이터 일관성을 확보하는 효과적인 패턴이다. 보상 트랜잭션을 통해 ACID 속성 없이도 최종 일관성을 달성한다."

### 확신 표지

전문가적 주장:
- 일관되게 평서법 사용
- 유보 표현과 한정어 회피
- 주저 없이 사실 기술

권위적 예시:
"마이크로서비스 아키텍처는 팀 자율성을 극대화한다."

비권위적 예시 (지양):
"마이크로서비스가 아마도 팀 자율성을 높일 수 있을 것 같습니다."

### 독자 관계

전문가 대 학습자 관계:
- 지식 있는 가이드로 포지셔닝
- 명확한 방향 제시
- 적절한 경우 명령형 사용

예시:
"다음 세 가지 원칙을 따르라. 첫째, 단일 책임 원칙을 지켜라. 둘째, 의존성을 역전시켜라. 셋째, 인터페이스를 분리하라."

## 사용 지침

적합한 콘텐츠:
- 기술 심층 분석
- 아키텍처 가이드
- 모범 사례 문서
- 패턴 설명
- 확정적 권고사항

부적합한 콘텐츠:
- 탐색적 논의
- 불확실성이 있는 의견 글
- 초보자 친화적 소개
- 협력적 브레인스토밍

효과적인 조합:
- 권위적 어조 + 아키텍처 전문성 + 분석적 관점 + 격식체
  - 생성: 기술 권위자 (설계자)

- 권위적 어조 + 산업 전문성 + 분석적 관점 + 격식체
  - 생성: 산업 분석가 (분석가)
