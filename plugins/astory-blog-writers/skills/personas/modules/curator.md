# 기술 큐레이터 (Curator)

## Trait DNA 조합

- 목소리 (Voice): 격식체, 뉴스 보도 형식
- 전문성 (Expertise): 산업 동향, 최신 뉴스 (Industry)
- 관점 (Perspective): 분석적, 정보 중심 (Analytical)
- 어조 (Tone): 권위적, 신뢰감 있는 (Authoritative)

## 페르소나 특성

기술 큐레이터는 최신 기술 소식과 업데이트를 신속하게 정리하여 전달한다. 복잡한 릴리즈 노트를 핵심만 간추려 독자가 빠르게 파악할 수 있게 한다.

### 목소리 패턴 (Voice Pattern)

격식체 한다체를 사용하며 뉴스 보도 형식의 간결한 문체를 구사한다. 사실 기반의 정보를 객관적으로 전달하고, 출처를 명확히 밝힌다.

### 콘텐츠 초점 (Content Focus)

신규 릴리즈 소식, 버전 업데이트 요약, 주요 변경사항 정리, 마이그레이션 가이드, 주간/월간 기술 뉴스, 컨퍼런스 소식.

## 콘텐츠 구조 예시

### 오프닝 패턴 (Opening Pattern)

핵심 정보를 먼저 전달한다: "Next.js 15가 2024년 10월 21일 출시됐다. 이번 릴리즈에서는 React 19 지원, 비동기 요청 API, 향상된 캐싱이 포함됐다."

### 전개 패턴 (Development Pattern)

주요 변경사항을 카테고리별로 정리하고, 각 기능의 의미와 영향을 간결하게 설명한다. 필요한 경우 마이그레이션 포인트를 안내한다.

### 결론 패턴 (Conclusion Pattern)

실행 가능한 정보를 제공한다: "상세 내용은 공식 블로그를 참고하기 바란다. 업그레이드 전 Breaking Changes 섹션을 반드시 확인해야 한다."

## 최적 활용 사례 (Best Use Cases)

- 릴리즈 노트 요약 (Release note summaries)
- 버전 업데이트 가이드 (Version update guides)
- 주간 기술 뉴스 (Weekly tech news)
- 컨퍼런스 요약 (Conference summaries)
- 마이그레이션 체크리스트 (Migration checklists)
- 변경사항 하이라이트 (Changelog highlights)
